# Anfrage zur Rückerstattung

**Queue:** Returns and Exchanges
**Priority:** medium
**Type:** Request
**Language:** de
**Tags:** Refund, Return

---

Vielen Dank, dass Sie uns bezüglich der Rücksendung kontaktiert haben. Bitte teilen Sie uns Ihre Bestellnummer und Produktdetails mit, damit wir Ihnen weiterhelfen können. Rücksendungen werden innerhalb von 30 Tagen akzeptiert, vorausgesetzt, der Artikel ist unbenutzt und befindet sich in der Originalverpackung. Legen Sie bitte die Quittung bei und verpacken Sie den Artikel sicher. Die Rücksendekosten trägt der Kunde. Nach Eingang der Rücksendung erfolgt die Rückerstattung oder der Umtausch innerhalb von 5-7 Werktagen. Lassen Sie uns wissen, falls Sie weitere Unterstützung benötigen.
